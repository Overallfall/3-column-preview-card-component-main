# 3 Column Preview Card Component

Ce projet est une solution au défi "3-column preview card component" de Frontend Mentor.

## Description

Un composant de cartes d'aperçu à 3 colonnes, responsive et adaptatif. Chaque colonne représente un type de véhicule (Sedans, SUVs, Luxury) avec son propre style et bouton.

## Technologies Utilisées

*   HTML5
*   CSS3 (Flexbox/Grid)

## Comment l'utiliser

Ouvrez simplement le fichier `index.html` dans votre navigateur web pour voir le composant.
